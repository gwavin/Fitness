Week 7 — Shoulder + Elbow Rehab with Barefoot Run

Building volume + shoulder strength
How to use this page
  • Click any exercise to open the full how‑to, cues, mistakes, and regressions/progressions.
  • Pain rule: 0–3/10 is acceptable. Stop for sharp/catching pain or if soreness >24h next day.
  • Week 7 progression: continue 3×6–8 @ RPE 6–7 for main lifts. Return to higher volume running (7 reps) with 2min intervals. Add more challenging shoulder exercises.
Week 7 Defaults
  • Goblet Squat, RDL: 3×6–8 @ RPE 6–7. Continue steady progression with form as priority.
  • Upper add‑ins: Chest‑Supported Row (neutral) 3×12–15; Push‑Up Plus progressing toward floor version; add light overhead movements.
  • Elbow: Eccentric wrist extension 3×10–15 + pronation/supination 3×15 + light grip work.
  • Run sessions: 7× (2min jog / 60s walk) between 5′ warm‑up/cool‑down (≈14 min jog total). Volume increase with same interval length.
  • Introduce light overhead reaches and more dynamic shoulder movements. Listen to shoulder feedback carefully.

Week 7 – Alternating Days (Weights ↔ Run) + Daily Shoulder Rehab

Continue gradual shoulder strengthening. Progress only when comfortable and pain-free. Use buttons to expand/collapse details.

Monday — Weights + Advanced Shoulder Work (~55–60 min)

Pendulum Swings — 30s each direction
  • Progress to larger circles and figure-8 patterns as comfortable.
Overhead Reach (wall slide) — 8 reps slow new movement
  • Back to wall; slide arms up wall as high as comfortable; pause 2s; lower slowly.
  • No forcing; stay within pain-free range. Focus on scapular upward rotation.
Band External Rotation (standing) — 2×15 light
  • Elbow at side; rotate outward; pause 1s; slow return. Increase reps slightly.
Goblet Squat — 3×6–8 @ RPE 6–7
  • Continue progression path; maintain excellent form throughout.
Romanian Deadlift (DB/KB) — 3×6–8
  • Hip hinge pattern mastery; controlled eccentric and concentric phases.
Chest‑Supported Row (neutral grip) — 3×12–15
  • Focus on scapular control and posterior deltoid activation.
Push‑Up Plus Progression — 3×8–12 volume increase
  • Add third set if handling current version well; work toward floor version.
Light Overhead Press (seated) — 2×8 very light new exercise
  • Seated with back support; very light DB; press up in pain-free range; slow and controlled.
  • Stop if any discomfort; this is exploratory only.
Eccentric Wrist Extension — 3×10–15
  • Can increase reps or very slightly increase weight if comfortable.
Grip Strengthening — 2×12
  • Stress ball or putty; 3s holds; slight volume increase.

Tuesday — Run + Calf/Shin Strength + Daily Shoulder Rehab (~50 min)

Short‑Foot Drill — 3×30s/side time increase
  • Arch activation; longer holds; maintain quality throughout duration.
Single-Leg Calf Raise — 3×8/side volume increase
  • If progressed to single-leg, add third set; otherwise continue double-leg progression.
Tibialis Raise — 3×18 volume increase
  • Back to wall; controlled dorsiflexion; slight rep increase.
Run – Week 7 Session A — 5' walk → 7× (2min jog / 60s walk) → 5' walk volume increase
  • Return to higher volume; maintain consistent pace; land softly throughout longer session.
  • If next‑morning stiffness >60 min, reduce to 6 intervals next session.
Scapular Setting — 5s ×15 volume increase
  • Gentle shrug → roll back → set blades down; focus on endurance.
Band Internal Rotation — 3×15 (light, 3s return) set increase
  • Add third set if handling well; maintain slow return phase.
Wall Angels — 3×12 set increase
  • Add third set; focus on range of motion improvements.
Push‑Up Plus (progression) — 2×12–15
  • Continue toward more challenging version; strong protraction.

Wednesday — Weights + Advanced Shoulder Work (~55–60 min)

Pendulum Swings — 30s each direction
  • Full range comfortable patterns; body-initiated movement.
Cat‑Cow — 12 reps slow
  • Thoracic mobility focus; breathe with movement; no rushing.
Y-T Raises (prone) — 2×10 light progression from Y-raises
  • Prone or incline; Y-position (arms up diagonally), then T-position (arms straight out).
  • Very light weights; pause 2s; focus on posterior deltoid and lower traps.
Goblet Squat — 3×6–8 @ RPE 6–7
  • Continue steady progression; perfect form maintained.
Romanian Deadlift (DB/KB) — 3×6–8
  • Hip hinge mastery; focus on glute and hamstring activation.
Chest‑Supported Row (neutral grip) — 3×12–15
  • Can experiment with slight grip width variations if comfortable.
Push‑Up Plus Progression — 3×8–12
  • Work toward floor version; maintain protraction quality.
Light Overhead Press (seated) — 2×8 very light
  • Continue exploration; stay within completely comfortable range.
Eccentric Wrist Extension — 3×12–15
  • Continue progression; maintain control focus.
Grip Strengthening — 2×12
  • Consistent grip work; 3s holds maintained.

Thursday — Run + Calf/Shin Strength + Daily Shoulder Rehab (~50 min)

Short‑Foot Drill — 3×30s/side
  • Maintain arch activation for full duration; quality over speed.
Single-Leg Calf Raise — 3×8/side (or continue progression)
  • Focus on control and full range of motion.
Tibialis Raise — 3×18
  • Controlled dorsiflexion; full range; pause briefly at top.
Run – Week 7 Session B — 5' walk → 7× (2min jog / 60s walk) → 5' walk
  • Consistent effort; focus on relaxed running form; soft landings.
  • Monitor morning stiffness; adjust volume if needed.
Scapular Setting — 5s ×15
  • Focus on endurance and consistency across all reps.
Band Internal Rotation — 3×15 (light, 3s return)
  • Maintain slow return phase; quality throughout all sets.
Wall Angels — 3×12
  • Work on improving range of motion gradually.
Push‑Up Plus (progression) — 2×12–15
  • Continue progression path; strong protraction at top.

Friday — Weights + Advanced Shoulder Work (~55–60 min)

Pendulum Swings — 30s each direction
  • Full comfortable range; relaxed arm hang.
Quadruped Shoulder Flexion — 10 reps/side
  • Hands/knees; reach forward/up; maintain spine neutral.
Band Pull-Aparts + External Rotation — 2×12 each combination exercise
  • Start with pull-apart, then rotate arms externally; return to start. Light resistance.
  • Targets multiple shoulder muscles; pause at end ranges.
Goblet Squat — 3×6–8 @ RPE 6–7
  • Maintain progression; form quality first.
Romanian Deadlift (DB/KB) — 3×6–8
  • Hip hinge pattern; controlled tempo throughout.
Chest‑Supported Row (neutral grip) — 3×12–15
  • Focus on scapular retraction and posterior deltoid.
Push‑Up Plus Progression — 3×8–12
  • Continue toward floor version; protraction quality maintained.
Light Overhead Press (seated) — 2×10 very light rep increase
  • If feeling good, add 2 reps; maintain perfect form and comfort.
Eccentric Wrist Extension — 3×12–15
  • Continue 3s lowering focus; slight progression available.
Grip Strengthening — 3×10 set increase
  • Add third set if handling well; maintain 3s holds.

Saturday — Run + Calf/Shin Strength + Daily Shoulder Rehab (~50 min)

Short‑Foot Drill — 3×30s/side
  • Quality arch activation; maintain throughout duration.
Single-Leg Calf Raise — 3×8/side
  • Control and full range priority; wall support as needed.
Tibialis Raise — 3×18
  • Wall support; controlled movement; pause at top.
Run – Week 7 Session C — 5' walk → 7× (2min jog / 60s walk) → 5' walk
  • Focus on rhythm and consistency; enjoy the longer session.
  • Listen to body; adjust if excessive stiffness develops.
Scapular Setting — 5s ×15
  • Endurance focus; consistent quality across all holds.
Band Internal Rotation — 3×15 (light, 3s return)
  • Slow return phase emphasis; maintain throughout all sets.
Wall Angels — 3×12
  • Work toward improved range; no forcing movement.
Push‑Up Plus (progression) — 2×12–15
  • Continue progression; strong protraction (plus) at top.

Sunday — Recovery Walk + Gentle Movement (~40 min)

Easy Walk — 25–35 min longer duration
  • Comfortable pace; enjoy nature; nasal breathing; focus on posture.
Gentle Stretch Routine — 8–10 min longer session
  • Pec stretch 30s; forearm stretch 30s; calf stretch 30s each leg; hip flexor stretch 30s each leg.
  • Add gentle shoulder rolls and neck stretches; thoracic spine rotation.
  • No aggressive stretching; focus on gentle lengthening and relaxation.
Scapular Setting — 5s ×15
  • Quality holds; focus on endurance and control.
Band Internal Rotation — 3×15 (very light)
  • Recovery-focused; smooth controlled movement.
Wall Angels — 3×12
  • Gentle range work; focus on smooth sliding motion.
Push‑Up Plus (easy version) — 2×12–15
  • Choose easier version for recovery day; maintain movement quality.
