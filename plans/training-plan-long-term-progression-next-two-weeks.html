Compact Training Plan

Low-intensity, long-horizon progression for durable running and resilient strength, built to fit a morning window ending by 06:45.

Core Rules

  • Keep most work easy and repeatable.
  • Progress only one variable at a time.
  • Prefer consistency over ambition.
  • Stop well short of grinding reps.
  • If symptoms are worse next morning, repeat or reduce.
Winning approach: 3 easy runs, 3 submaximal strength sessions, small progressions, low drama.

3-Year Direction

  • 0-3 months: rebuild habit and tissue tolerance
  • 3-9 months: build easy running base and stable strength
  • 9-18 months: consolidate into a dependable weekly rhythm
  • 18-36 months: refine toward stronger bench, more robust running, or both

Running Thread

Current base: gentle (90 sec run + 60 sec walk) x 5

Progression order

  • Consistency
  • Longer run intervals
  • More total running time
  • Less walk recovery
  • Harder surfaces later
  • Speed much later

Strength Thread

Current warm-up circuit x 3:

  • Bodyweight squat x 5
  • Push-up x 5
  • Hip hinge x 5
  • Dead hang x 5
  • Kettlebell swing x 5

Bench approach

  • Top set of 5
  • Then 2 back-off sets of 5 at 85-90%
  • Keep rows and hangs in permanently

Week 1

Run A

Monday

  • Warm-up: 3-5 min walk, ankle circles, calf raises, marching, 2 gentle jogs
  • Main: (90 sec run + 60 sec walk) x 6
  • Cool-down: 3-5 min walk, light calf stretch, knee-to-wall mobility
Strength A

Tuesday

  • Warm-up circuit x 3
  • Bench: 20x5, 40x5, 50x5, 65x5, then 57.5-60x5x2
  • Pull: inverted row 3x5-8 or DB row 3x8 each side
  • Hang: 2 x 10-20 sec
Run B

Wednesday

  • Repeat (90 sec run + 60 sec walk) x 6
Strength B

Thursday

  • Warm-up circuit x 3
  • Bench: 20x5, 40x5, 50x5, 65x5, then 60x5x2
  • Lower support: KB RDL 2-3x8 or goblet squat 2-3x6-8
  • Pull: 3 sets
  • Optional plank: 2 x 20-30 sec
Run C

Friday

  • If week feels calm: (100 sec run + 60 sec walk) x 6
  • If not: repeat (90 + 60) x 6
Strength C

Saturday

  • If week felt easy: 20x5, 40x5, 50x5, 67.5x5, then 57.5-60x5x2
  • If caution preferred: repeat 65x5 with back-offs
  • Pull: 3 sets
  • Lower support: 2 easy sets
  • Hang: 2 x 15-20 sec
Recovery

Sunday

  • Easy walk, shoulder mobility, ankle mobility

Week 2

Run A2

Monday

  • If Week 1 was smooth: (100 sec run + 60 sec walk) x 6
  • If not: repeat last successful level
Strength A2

Tuesday

  • Warm-up circuit x 3
  • If 65 still feels best: 20x5, 40x5, 50x5, 67.5x5, then 60x5x2
  • If 67.5 felt sticky: stay at 65x5
  • Pull: 3 sets
  • Optional scap push-up or push-up plus: 2x8 if comfortable
Run B2

Wednesday

  • Repeat (100 sec run + 60 sec walk) x 6 or previous successful level
Strength B2

Thursday

  • Warm-up circuit x 3
  • Bench: same loading as Tuesday
  • Lower support: goblet squat or KB RDL, 2-3 easy sets
  • Pull: 3 sets
  • Hang: 2 sets
Run C2

Friday

  • If fortnight is calm: (2 min run + 60 sec walk) x 6
  • If anything is grumbly: stay at (100 sec + 60 sec) x 6
Strength C2

Saturday

  • Bench options: repeat 67.5x5, or 70x5 only if everything feels extremely good
  • Most conservative choice: 67.5x5
  • Pull: 3 sets
  • Lower support: 2 easy sets
  • Hang: 2 sets
Recovery

Sunday

  • Walk, mobility, no need to make up missed work

Morning Timing

Run days

  • 05:45 or 06:00: up and dressed
  • 3-5 min warm-up
  • 20-25 min main set
  • 5 min cool-down
  • Done before 06:45

Strength days

  • 5 min setup
  • 10 min warm-up circuit
  • 15-20 min bench work
  • 10-15 min rows, lower support, hangs

What to Avoid

  • Speed work
  • Hill work
  • Testing continuous running too early
  • Grinding bench reps
  • Adding lots of overhead work
  • Pushing running and lower-body lifting hard together
After this fortnight: only advance if tissues are calm. The plan should feel almost boring, in a good way.

Possible Next Running Ladder

  • 2 min run / 1 min walk x 6
  • 3 min run / 1 min walk x 5
  • 4 min run / 90 sec walk x 5
  • 5 min run / 90 sec walk x 4-5
  • Later: longer continuous easy running

Bottom Line

  • Keep everything easy enough to repeat.
  • Let months accumulate.
  • Progress in tiny steps.
  • Protect shoulder, calf, foot and ankle tissues.
  • Stay patient and keep training enjoyable.
