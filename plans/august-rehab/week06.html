Week 6 — Shoulder + Elbow Rehab with Barefoot Run

Longer intervals + shoulder strengthening
How to use this page
  • Click any exercise to open the full how‑to, cues, mistakes, and regressions/progressions.
  • Pain rule: 0–3/10 is acceptable. Stop for sharp/catching pain or if soreness >24h next day.
  • Week 6 progression: maintain 3×6–8 @ RPE 6–7 for main lifts. Running intervals get longer (2min) but volume decreases slightly. Add more dynamic shoulder exercises.
Week 6 Defaults
  • Goblet Squat, RDL: 3×6–8 @ RPE 6–7. Continue progression: +1 rep or micro‑load as tolerated.
  • Upper add‑ins: Chest‑Supported Row (neutral) 3×10–15; Push‑Up Plus progressing toward floor version 2–3×8–12.
  • Elbow: Eccentric wrist extension 3×10–12 (3s down) + pronation/supination 2–3×15. Add light grip strengthening if comfortable.
  • Run sessions: 6× (2min jog / 60s walk) between 5′ warm‑up/cool‑down (≈12 min jog total). Longer intervals, fewer reps.
  • Introduce light resistance band exercises for shoulders. Maintain neutral grips where possible.

Week 6 – Alternating Days (Weights ↔ Run) + Daily Shoulder Rehab

Progress shoulder exercises gradually. Maintain pain-free ranges only. Use the buttons below to expand/collapse details.

Monday — Weights + Elbow Care (~50–55 min)

Pendulum Swings — 30s each direction
  • Lean forward, hand on a table/chair. Let the affected arm hang completely relaxed.
  • Start tiny circles from body momentum. Progress to figure-8 patterns if comfortable.
Open Books (T‑spine) — 10 reps/side
  • Side‑lying; hips/knees 90°. Reach top hand up, then open toward the floor behind you.
  • Breathe out into the range; focus on thoracic rotation; pause at end range.
Band External Rotation (standing) — 2×12 light new exercise
  • Band at elbow height; elbow at side at 90°; rotate arm outward; pause 1s; slow return.
  • Keep elbow glued to side; no compensation through body. Very light resistance.
Goblet Squat — 3×6–8 @ RPE 6–7 • 3‑1‑1 tempo
  • KB/DB at chest; feet shoulder‑width; brace; sit between hips; chest tall; knees track over mid‑foot.
Romanian Deadlift (DB/KB) — 3×6–8 light–moderate
  • Soft knees; hinge hips back; spine neutral; weights skim thighs; controlled movement.
Chest‑Supported Row (neutral grip) — 3×10–15 volume increase
  • Bench 30–45°; chest down; focus on squeezing blades together; pause 1s; 2–3s negative.
Push‑Up Plus Progression — 2–3×8–12 progress toward floor
  • If incline version is mastered, try knee push‑up plus or full push‑up plus. Focus on protraction at top.
  • Quality over quantity; perfect form essential.
Eccentric Wrist Extension — 3×10–12 (3s eccentric)
  • Light weight; forearm supported; slow 3s lowering; can increase reps slightly if comfortable.
Pronation/Supination — 2–3×15
  • Light hammer/weight; smooth rotation; full pain-free range.

Tuesday — Run + Calf/Shin Prep + Daily Shoulder Rehab (~50 min)

Short‑Foot Drill — 3×25s/side time increase
  • Lift arch gently; avoid toe curling; breathe normally; focus on maintaining position longer.
Single-Leg Calf Raise — 2×8/side progression
  • Hold wall for balance; full range; 2s up/down; rise over big/second toe. Progress from double-leg version.
  • If too challenging, continue double-leg version 3×15.
Tibialis Raise — 3×15 volume increase
  • Back to wall; heels down; lift toes toward shins; control the lower.
Run – Week 6 Session A — 5' walk → 6× (2min jog / 60s walk) → 5' walk longer intervals
  • Cadence ~170–180; land under center; quiet feet; focus on rhythm for longer intervals.
  • If next‑morning calf/Achilles stiffness >60 min, reduce to Week 5 volume.
Scapular Setting — 5s ×12
  • Gentle shrug → roll back → set blades down into back pockets; neck relaxed; breathe.
Band Internal Rotation — 2×15 (very light, 3s return) volume increase
  • Anchor at elbow height; elbow glued to side at 90°; pull forearm toward belly; pause; slow 3s return.
Wall Angels — 2×12 volume increase
  • Back/head/butt to wall; ribs down; slide up only to clean range; focus on smooth movement.
Push‑Up Plus (incline or floor progression) — 1×12–15
  • Progress toward lower incline or knee version. At top, protract shoulder blades strongly.

Wednesday — Weights + Elbow Care (~50–55 min)

Pendulum Swings — 30s each direction
  • Let arm hang relaxed; small circles progressing to figure-8 if comfortable.
Cat‑Cow — 10 reps slow
  • Hands/knees; round up like a cat, then arch gently. Breathe with movement.
Y-Raises (prone or incline) — 2×8 very light new exercise
  • Lie face down or chest on incline bench; arms in Y position; lift slightly; pause 2s; lower slow.
  • Very light weights or no weights; focus on posterior deltoid and lower traps.
Goblet Squat — 3×6–8 @ RPE 6–7
  • Continue progression; maintain excellent form throughout all reps.
Romanian Deadlift (DB/KB) — 3×6–8
  • Hip hinge pattern; neutral spine; controlled tempo.
Chest‑Supported Row (neutral grip) — 3×10–15
  • Focus on scapular control; vary grip width slightly if comfortable.
Push‑Up Plus Progression — 2–3×8–12
  • Continue progressing toward more challenging variation; maintain protraction quality.
Eccentric Wrist Extension — 3×10–12 (3s eccentric)
  • Maintain slow 3s lowering; can slightly increase weight if form remains perfect.
Light Grip Strengthening — 2×10 new addition
  • Light stress ball or putty; squeeze and hold 3s; release slowly. Pain-free range only.

Thursday — Run + Calf/Shin Prep + Daily Shoulder Rehab (~50 min)

Short‑Foot Drill — 3×25s/side
  • Lift arch gently; maintain position; breathe normally.
Single-Leg Calf Raise — 2×8/side (or continue double-leg 3×15)
  • Progress only if double-leg version feels easy; maintain quality.
Tibialis Raise — 3×15
  • Back to wall; controlled toe raises; full range of motion.
Run – Week 6 Session B — 5' walk → 6× (2min jog / 60s walk) → 5' walk
  • Focus on consistent pace throughout 2-minute intervals; cadence ~170–180.
  • If stiffness >60 min next morning, reduce interval length next session.
Scapular Setting — 5s ×12
  • Gentle shrug → roll back → set blades down; focus on holding position cleanly.
Band Internal Rotation — 2×15 (very light, 3s return)
  • Elbow at side; pull toward belly; pause; slow return. Perfect form.
Wall Angels — 2×12
  • Back to wall; slide arms up only to pain-free range; smooth motion.
Push‑Up Plus (progression) — 1×12–15
  • Continue toward more challenging version; protract blades at top.

Friday — Weights + Elbow Care (~50–55 min)

Pendulum Swings — 30s each direction
  • Relaxed arm hang; body-initiated movement; progress to figure-8 pattern.
Quadruped Shoulder Flexion — 8 reps/side
  • Hands/knees; reach arm forward/up; hold 2s; maintain neutral spine.
Band Pull-Aparts — 2×12 light new exercise
  • Arms straight, band at chest level; pull apart squeezing shoulder blades; pause 2s; slow return.
  • Light resistance; focus on posterior deltoid and rhomboids.
Goblet Squat — 3×6–8 @ RPE 6–7
  • Maintain progressive overload; perfect form priority.
Romanian Deadlift (DB/KB) — 3×6–8
  • Hip hinge; neutral spine; controlled movement pattern.
Chest‑Supported Row (neutral grip) — 3×10–15
  • Scapular retraction focus; pause at top position.
Push‑Up Plus Progression — 2–3×8–12
  • Work toward floor version if ready; maintain protraction quality.
Eccentric Wrist Extension — 3×10–12 (3s eccentric)
  • Continue focus on controlled lowering phase.
Light Grip Strengthening — 2×10
  • Stress ball; squeeze 3s holds; pain-free range only.

Saturday — Run + Calf/Shin Prep + Daily Shoulder Rehab (~50 min)

Short‑Foot Drill — 3×25s/side
  • Arch activation; avoid toe gripping; steady breathing.
Single-Leg Calf Raise — 2×8/side (or double-leg 3×15)
  • Quality over progression; maintain control throughout range.
Tibialis Raise — 3×15
  • Wall support; controlled dorsiflexion; pause at top briefly.
Run – Week 6 Session C — 5' walk → 6× (2min jog / 60s walk) → 5' walk
  • Maintain relaxed effort; focus on landing softly; rhythm consistency.
  • Stop early if calves/Achilles complain; adjust volume as needed.
Scapular Setting — 5s ×12
  • Gentle shrug → roll back → set blades down; breathe normally.
Band Internal Rotation — 2×15 (very light, 3s return)
  • Anchor at elbow height; controlled rotation; pause and slow return.
Wall Angels — 2×12
  • Back to wall; pain-free range only; focus on smooth sliding motion.
Push‑Up Plus (progression) — 1×12–15
  • Progress appropriately; strong protraction (plus) at top position.

Sunday — Recovery Walk + Daily Shoulder Rehab (~35 min)

Easy Walk — 20–30 min
  • Comfortable pace; nasal breathing; enjoy the movement and fresh air.
Gentle Stretch — Pec, forearm, calf, & hip flexor, 30s each add hip flexor
  • Gentle lengthening; no aggressive stretching; hold each position comfortably.
Scapular Setting — 5s ×12
  • Gentle shrug → roll back → set blades down; relaxed neck.
Band Internal Rotation — 2×15 (very light, 3s return)
  • Elbow at side; controlled movement; focus on return phase.
Wall Angels — 2×12
  • Back to wall; slide up to comfortable range; smooth breathing.
Push‑Up Plus (progression) — 1×12–15
  • Continue progression path; protract blades cleanly at top.
