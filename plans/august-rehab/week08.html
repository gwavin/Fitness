Longer intervals + functional strength
How to use this page
  • Click any exercise to open the full how‑to, cues, mistakes, and regressions/progressions.
  • Pain rule: 0–3/10 is acceptable. Stop for sharp/catching pain or if soreness >24h next day.
  • Week 8 progression: continue 3×6–8 @ RPE 6–7 for main lifts. Running progresses to 3-minute intervals with longer rest. Add functional shoulder movements.
Week 8 Defaults
  • Goblet Squat, RDL: 3×6–8 @ RPE 6–7. Continue steady progression; consider slight load increases if form perfect.
  • Upper add‑ins: Chest‑Supported Row (neutral) 3×12–15; Push‑Up Plus working toward floor version; light functional overhead work.
  • Elbow: Eccentric wrist extension 3×12–15 + pronation/supination 3×15–18 + grip strengthening.
  • Run sessions: 6× (3min jog / 90s walk) between 5′ warm‑up/cool‑down (≈18 min jog total). Longer intervals, longer rest.
  • Introduce more dynamic and functional shoulder movements. Begin preparing for return to normal activities.

Week 8 – Alternating Days (Weights ↔ Run) + Daily Shoulder Rehab

Focus on functional strength and endurance. Begin transitioning toward normal movement patterns. Use buttons to expand/collapse details.

Monday — Weights + Functional Shoulder Work (~60 min)

Arm Circles — 10 forward, 10 backward progression from pendulums
  • Standing; small controlled circles progressing to larger; both directions.
  • Active movement but stay within comfortable range; no forcing.
Overhead Reach (wall slide) — 10 reps slow
  • Back to wall; slide arms up as high as comfortable; work on range improvement.
Band External Rotation — 3×15 light volume increase
  • Add third set; elbow at side; rotate outward; pause 1s; slow return.
Goblet Squat — 3×6–8 @ RPE 6–7
  • Consider small load increase if hitting 8 reps easily with perfect form.
Romanian Deadlift (DB/KB) — 3×6–8
  • Continue progressive overload; hip hinge mastery maintained.
Chest‑Supported Row (neutral + underhand) — 3×12–15 grip variation
  • Try alternating grip styles between sets if comfortable; focus on scapular control.
Push‑Up Plus (floor progression) — 3×8–12
  • Work toward full floor push-up plus; maintain protraction quality.
Light Overhead Press (seated to standing) — 2×10–12 light progression
  • If seated version comfortable, try standing with back support. Very light weight.
  • Focus on pain-free range; stop if any discomfort.
Farmer's Walk — 2×20 meters light new functional exercise
  • Light weights; walk with good posture; engage core and shoulders for stability.
  • Functional shoulder and grip strengthening.
Eccentric Wrist Extension — 3×12–15
  • Continue 3s lowering; can slightly increase weight if comfortable.
Grip Strengthening — 3×12
  • Stress ball or putty; 3s holds; consistent form.

Tuesday — Run + Lower Leg Strength + Daily Shoulder Rehab (~55 min)

Short‑Foot Drill — 4×30s/side volume increase
  • Add fourth set; arch activation endurance; quality maintained.
Single-Leg Calf Raise — 3×10/side rep increase
  • If progressed to single-leg, add 2 reps; full range and control.
Tibialis Raise — 3×20 rep increase
  • Wall support; controlled dorsiflexion; pause briefly at top.
Run – Week 8 Session A — 5' walk → 6× (3min jog / 90s walk) → 5' walk longer intervals + rest
  • Longer intervals require pacing; start conservatively; build rhythm for 3-minute bouts.
  • If intervals feel too challenging, reduce to 2.5 minutes; progression is gradual.
Scapular Setting + Hold — 8s ×12 longer holds
  • Extend hold time to 8s; focus on endurance and control.
Band Internal + External Rotation — 2×12 each direction combination
  • Internal rotation followed immediately by external rotation; light resistance.
Wall Angels + Overhead Reach — 3×10 combination
  • Wall angels ending with overhead reach; work on range of motion.
Push‑Up Plus (floor progression) — 2×10–15
  • Continue progression; strong protraction at top position.

Wednesday — Weights + Functional Shoulder Work (~60 min)

Arm Circles — 10 each direction
  • Active controlled circles; full comfortable range.
Cat‑Cow + Thoracic Extension — 12 reps enhanced version
  • Standard cat-cow adding thoracic extension; sit back toward heels briefly.
  • Extra thoracic mobility focus; breathe with movement.
Y-T-W Raises (prone) — 2×8 each position light full sequence
  • Y-position, then T-position, then W-position (arms bent like goal posts).
  • Very light weights; 2s pause at top; focus on posterior deltoid and lower traps.
Goblet Squat — 3×6–8 @ RPE 6–7
  • Maintain progression; consider load increase if form perfect.
Romanian Deadlift (DB/KB) — 3×6–8
  • Hip hinge pattern; continue progressive overload path.
Chest‑Supported Row (neutral + underhand) — 3×12–15
  • Vary grip; focus on scapular retraction and posterior deltoid.
Push‑Up Plus (floor progression) — 3×8–12
  • Continue toward floor version; protraction quality maintained.
Light Overhead Press (standing) — 2×10–12 light
  • Progress to standing if comfortable; back support available; very light weight.
Farmer's Walk — 2×30 meters light
  • Extend distance slightly; maintain posture and shoulder stability.
Eccentric Wrist Extension — 3×12–15
  • Continue controlled lowering; slight progression available.
Grip Strengthening + Finger Extension — 3×10 each add finger extension
  • Grip work followed by finger extension with band; balanced hand strengthening.

Thursday — Run + Lower Leg Strength + Daily Shoulder Rehab (~55 min)

Short‑Foot Drill — 4×30s/side
  • Quality arch activation; maintain throughout all sets.
Single-Leg Calf Raise — 3×10/side
  • Control and full range; wall support as needed.
Tibialis Raise — 3×20
  • Full range dorsiflexion; controlled movement pattern.
Run – Week 8 Session B — 5' walk → 6× (3min jog / 90s walk) → 5' walk
  • Consistent pacing for 3-minute intervals; focus on rhythm and form.
  • Monitor effort level; should feel sustainable throughout all intervals.
Scapular Setting + Hold — 8s ×12
  • Endurance focus; 8s holds maintained throughout.
Band Internal + External Rotation — 2×12 each direction
  • Smooth transitions; light resistance; quality control.
Wall Angels + Overhead Reach — 3×10
  • Range of motion work; gradual improvement focus.
Push‑Up Plus (floor progression) — 2×10–15
  • Continue progression path; maintain quality over quantity.

Friday — Weights + Functional Shoulder Work (~60 min)

Arm Circles + Cross-Body Stretch — 10 circles + 20s stretch combination warm-up
  • Arm circles followed by gentle cross-body shoulder stretch.
  • Comprehensive shoulder preparation; no aggressive stretching.
Quadruped Shoulder Flexion + Lateral Reach — 8 reps each direction enhanced version
  • Forward reach, then lateral reach (arm out to side); maintain neutral spine.
  • Multi-directional shoulder mobility and stability.
Band Complex — 2×10 (Pull-Apart + External Rotation + High Pull) combination exercise
  • Pull-apart, then external rotation, then high pull (elbows up). Light resistance.
  • Comprehensive posterior shoulder strengthening.
Goblet Squat — 3×6–8 @ RPE 6–7
  • Continue steady progression; form quality maintained.
Romanian Deadlift (DB/KB) — 3×6–8
  • Hip hinge mastery; controlled tempo throughout range.
Chest‑Supported Row (varied grips) — 3×12–15
  • Experiment with grip variations; focus on scapular control.
Push‑Up Plus (floor progression) — 3×8–12
  • Work toward full floor version; maintain protraction strength.
Light Overhead Press (standing) — 3×8–10 light volume increase
  • Add third set if comfortable; maintain perfect form; very light weight.
Farmer's Walk + Suitcase Carry — 20m each variation
  • Farmer's walk (both hands) then suitcase carry (one side). Light weight.
  • Core and shoulder stability challenge.
Eccentric Wrist Extension — 3×12–15
  • Continue controlled lowering focus; slight progression available.
Grip Strengthening + Finger Extension — 3×12 each
  • Balanced hand strengthening; grip and extension work.

Saturday — Run + Lower Leg Strength + Daily Shoulder Rehab (~55 min)

Short‑Foot Drill — 4×30s/side
  • Consistent arch activation; quality throughout all sets.
Single-Leg Calf Raise — 3×10/side
  • Full range control; wall support available as needed.
Tibialis Raise — 3×20
  • Wall support; controlled dorsiflexion; brief pause at top.
Run – Week 8 Session C — 5' walk → 6× (3min jog / 90s walk) → 5' walk
  • Focus on consistent effort; soft landings; enjoy the longer session.
  • Listen to body signals; adjust if excessive fatigue or stiffness.
Scapular Setting + Hold — 8s ×12
  • Focus on endurance and consistent quality throughout.
Band Internal + External Rotation — 2×12 each direction
  • Smooth controlled movement; light resistance maintained.
Wall Angels + Overhead Reach — 3×10
  • Range of motion focus; gradual improvement over time.
Push‑Up Plus (floor progression) — 2×10–15
  • Continue progression; strong protraction at top position.

Sunday — Active Recovery + Comprehensive Stretching (~45 min)

Easy Walk — 30–40 min
  • Longer recovery walk; enjoy nature; nasal breathing; focus on posture and relaxation.
Comprehensive Stretch Routine — 10–12 min full body
  • Neck stretch 20s each direction; shoulder rolls 10 each direction.
  • Pec stretch 30s; cross-body shoulder stretch 30s each arm.
  • Forearm stretch 30s each; thoracic spine rotation 10 each direction.
  • Calf stretch 30s each leg; hip flexor stretch 30s each leg.
  • Gentle spinal twist 20s each side; child's pose 30s.
  • Focus on relaxation and gentle lengthening; no aggressive stretching.
Scapular Setting + Hold — 8s ×12
  • Recovery-focused; quality holds with relaxed approach.
Band Internal + External Rotation — 2×12 each (very light)
  • Very light resistance; smooth movement; recovery emphasis.
Wall Angels (gentle) — 2×12
  • Gentle range work; focus on smooth sliding motion.
Push‑Up Plus (easy version) — 1×15
  • Choose easier version for recovery; maintain movement pattern.
