Week 9 — Shoulder + Elbow Rehab with Barefoot Run

Volume build + advanced shoulder work
How to use this page
  • Click any exercise to open the full how‑to, cues, mistakes, and regressions/progressions.
  • Pain rule: 0–3/10 is acceptable. Stop for sharp/catching pain or if soreness >24h next day.
  • Week 9 progression: continue 3×6–8 @ RPE 6–7 for main lifts. Running volume increases to 7 reps with 3-minute intervals. Add more advanced shoulder strengthening.
Week 9 Defaults
  • Goblet Squat, RDL: 3×6–8 @ RPE 6–7. Continue progression; may consider transitioning to barbell versions if shoulder allows.
  • Upper add‑ins: Chest‑Supported Row 3×12–15; Push‑Up Plus (floor) 3×8–12; light pressing movements.
  • Elbow: Eccentric wrist extension 3×12–15 + pronation/supination 3×15–20 + grip strengthening + finger extension.
  • Run sessions: 7× (3min jog / 90s walk) between 5′ warm‑up/cool‑down (≈21 min jog total). Higher volume with 3-min intervals.
  • Focus on shoulder stability under load and dynamic movements. Begin integration of multi-plane movements.

Week 9 – Alternating Days (Weights ↔ Run) + Daily Shoulder Rehab

Monday — Advanced Strength + Shoulder Integration (~60–65 min)

Dynamic Arm Swings — 12 each direction progression from circles
  • Forward/backward swings, then cross-body swings; controlled but dynamic.
  • Prepare shoulder for loaded movements; full comfortable range.
Wall Handstand Hold — 15–20s new challenge
  • Feet up wall, hands on ground; hold position; focus on shoulder stability.
  • Regression: incline hands on bench. Only if shoulders feel strong and stable.
  • Skip if any shoulder discomfort; this is advanced.
Band External Rotation + Overhead Reach — 3×12 combination exercise
  • External rotation followed immediately by overhead reach; light resistance throughout.
Goblet Squat (consider barbell transition) — 3×6–8 @ RPE 6–7
  • If shoulder stable with load, consider bodyweight back squat exploration.
  • Only progress if completely comfortable; goblet version still excellent.
Romanian Deadlift (DB/KB → Barbell consideration) — 3×6–8
  • Continue progression; may explore barbell version if grip and shoulder stable.
Chest‑Supported Row + Single-Arm Variation — 3×10–12 bilateral + 2×8 each arm unilateral addition
  • Bilateral rows followed by single-arm work; challenges stability differently.
Push‑Up Plus (floor) — 3×10–15 floor version goal
  • Work toward full floor push-up plus; strong protraction maintained.
Overhead Press (seated → standing) — 3×8–12 light to moderate
  • Progress to standing if comfortable; can slightly increase weight if form perfect.
  • Pain-free range priority; stop at any discomfort.
Loaded Carry Complex — 2×40m increased challenge
  • 20m farmer's walk + 20m suitcase carry (switch sides). Light to moderate weight.
  • Core and shoulder stability under load.
Advanced Wrist + Grip Work — 3×12 each exercise
  • Eccentric wrist extension + pronation/supination + grip squeeze + finger extension.
  • Comprehensive forearm strengthening complex.

Tuesday — Extended Run + Lower Leg Strength + Daily Shoulder Rehab (~55–60 min)
