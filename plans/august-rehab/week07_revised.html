Week 7 — Shoulder + Elbow Rehab with Barefoot Run

Consolidate → add unilateral + anti‑rotation
Why this week?
  • From Week 6: you handled 3×6–8 @ RPE 6–7 on main lifts and 6× (2′ jog / 60″ walk). This week keeps intensity but adds unilateral work (split squat, single‑leg hinge), anti‑rotation (suitcase carry/pallof), and slightly higher run volume for resilience.
  • Green‑light rule: Only progress a session if next‑day symptoms ≤3/10 and no increase in night pain. If not, repeat the last clean dose.
  • Pain rules (shoulder/elbow): 0–3/10 acceptable during; stop on sharp/catching pain; DOMS should resolve ≤24 h.
Week 7 Defaults
  • Main lower lifts: stay at 3×6–8 @ RPE 6–7. Add a unilateral variant each weights day (split squat or step‑back lunge; SL‑RDL pattern).
  • Shoulder: progress to scaption raises and band external rotation with 3‑sec returns; keep push‑up plus at a manageable incline.
  • Elbow: continue eccentric wrist extension 3×10–12 (3s down) + pronation/supination 2–3×12–15; keep a 20–30s isometric “calm set” for flares.
  • Run: 7× (2′ jog / 60″ walk) between 5′ warm‑up/cool‑down. If next‑morning calf/Achilles stiffness >60 min, repeat volume.
  • Anti‑rotation: add Pallof press or Suitcase carry each weights day (20–40 m or 8–10 controlled presses/side).

Week 7 – Alternating Days (Weights ↔ Run) + Daily Shoulder/Elbow Care

Use the buttons to expand/collapse details. Progress range, then reps, then load.

Monday — Weights + Unilateral Focus (~50–55 min)

Open Books (T‑spine) — 6/side
  • Side‑lying; breathe out into the rotation; keep knees stacked.
Goblet Squat — 3×6–8 @ RPE 6–7 • 3‑1‑1 tempo
  • Brace; sit between hips; heels down; 1s pause at bottom.
DB Romanian Deadlift — 3×6–8 light‑moderate
  • Hips back; soft knees; neutral spine; lats packed. Straps ok if elbow talks.
Rear‑Foot Elevated Split Squat — 2×6–8/side new unilateral
  • Torso tall; knee tracks over mid‑foot; small ROM if needed.
  • Reduce height or swap to step‑back lunge if anterior knee/shoulder complains.
Band External Rotation — 2–3×12 (3s return)
  • Elbow at 90° glued to side (towel roll). Rotate out; slow return.
Scaption Raise (thumbs up) — 2×10 light
  • Arms ~30–45° front of body (“Y” path); stop before shrugging.
Pallof Press — 2×8–10/side (2s hold)
  • Band at sternum height; resist rotation; ribs down; slow tempo.
Eccentric Wrist Extension — 3×10–12 (3s down) + Calm‑Set Isometric 20–30s
  • Help up with other hand; lower against light/moderate load; pain ≤3/10.

Tuesday — Barefoot Run + Foot/Calf Prep (~30–35 min)

Short‑Foot Drill — 3×20s/side
  • Lift arch without toe curl; breathe; no cramps.
Double‑Leg Calf Raise — 3×12 slow
  • Rise over big/second toe; 1s pause; 2–3s down.
Tibialis Raise — 2×15
  • Back to wall; heels down; toes to shins; slow.
Run — Session A — 5′ walk → 7× (2′ jog / 60″ walk) → 5′ walk
  • Cadence 170–180; land under center; quiet feet; posture tall with slight forward lean from ankles.
  • If next‑morning calf/Achilles stiffness >60 min, repeat this session before progressing.
Elbow Pronation/Supination — 2×12–15 light
  • DB like a hammer; rotate slow; elbow glued to ribs.

Wednesday — Weights + Single‑Leg Hinge (~50–55 min)

Pendulum Swings — 30s each direction
  • Relaxed shoulder; body initiates motion.
Goblet Squat — 3×6–8 @ RPE 6–7
  • Match or +1 rep vs Monday if clean.
DB RDL — 3×6–8
  • Hips back; neutral; straps ok.
Supported Single‑Leg RDL — 2×6–8/side new unilateral
  • One hand to wall/bench for balance; hinge small range first.
Incline Push‑Up Plus — 2–3×8–12
  • Protract at top without shrug; lower the incline only if pain‑free.
Band External Rotation — 2×12 (3s return)
  • Towel roll; ribs down; slow.
Suitcase Carry — 2×20–40 m/side
  • Stand tall; shoulder blades down/back; don’t lean.
Eccentric Wrist Extension — 3×10–12
  • 3s down; pain ≤3/10.

Thursday — Barefoot Run + Foot/Calf Prep (~30–35 min)

Short‑Foot — 3×20s/side
  • Gentle arch lift.
Double‑Leg Calf Raise — 3×12
  • Full range; slow.
Tibialis Raise — 2×15
  • Toes to shins; slow.
Run — Session B — 5′ walk → 7× (2′ jog / 60″ walk) → 5′ walk
  • Match Tuesday. If more than easy, keep it easy—save progress for Week 8.
Elbow Pronation/Supination — 2×12–15

Friday — Weights + Integration (~50–55 min)

Open Books — 6/side
Goblet Squat — 3×6–8 @ RPE 6–7
DB RDL — 3×6–8
Step‑Back Lunge — 2×6–8/side
  • Small ROM; control descent; neutral grip to spare elbow.
Push‑Up Plus (incline or floor) — 2–3×8–12
Band External Rotation — 2×12 (3s return)
Pallof Press — 2×8–10/side (2s hold)
Eccentric Wrist Extension — 3×10–12 + Calm‑Set Isometric 20–30s

Saturday — Barefoot Run (Optional C) or Easy Walk

Run — Session C (optional) — 5′ walk → 6× (2′ jog / 60″ walk) → 5′ walk
  • Only if Tue/Thu were green‑light and you feel springy. Otherwise, 10–20′ easy walk.
Gentle Mobility — 5–10′
  • Pecs & forearms 20–30s each; no aggressive stretching.

Sunday — Recovery

Easy Walk — 10–20′
Scapular Setting — 5s ×10
Notes
  • If any session produced >3/10 pain that lingered, mark it and repeat the last clean dose before progressing.
  • Progression order next week: range → reps → load. Avoid jumping two variables at once.
Exit criteria for Week 8
  • Push‑up plus at chosen incline is crisp for 3×10 with no shoulder pinch.
  • Eccentric wrist extension 3×12 with pain ≤2/10 during, none after 24 h.
  • All three runs completed with next‑morning calf/Achilles stiffness ≤30 min.
