Week 5 Defaults
  • Squat (choose one): Back Squat if shoulders are happy 3×6–8 @ RPE 6–7 • OR stay Goblet/FFESS same sets/reps/effort.
  • Hinge: RDL 3×8 light–moderate, perfect technique.
  • Upper: Chest‑Supported Row 3×10–12 • Neutral‑Grip DB Press (slight incline) 3×8–10.
  • Cuff: Band ER 2–3×12 + IR 3×10–12 slow 3s returns.
  • Elbow: Wrist‑extensor HSR 3×10–12 (2–3s up/3s down). Keep a single isometric calm set only if flared.
  • Run sessions: 6–8× (2:00 jog / 1:00–2:00 walk) between 5′ warm‑up/cool‑down. If next‑morning calf/Achilles stiffness >60 min, repeat same volume.
  • Grip neutral where possible; straps allowed on hinge work if elbow talks.

Week 5 – Alternating Days (Weights ↔ Run) + Daily Shoulder Rehab

Progress only when next‑day symptoms are ≤3/10. Use the buttons below to expand/collapse details.

Monday — Weights + Elbow HSR (~45–50 min)

Open Books (T‑spine) — 6 reps/side
  • Side‑lying; hips/knees 90°. Reach top hand to ceiling, then rotate open toward floor behind you while knees stay stacked. Exhale into range.
  • Stop before any shoulder pinch.
Squat — choose one — 3×6–8 @ RPE 6–7 • 3‑1‑1 tempo
  • Back Squat: Hands comfortable width; set blades down/back; bar on traps; ribs down. 3s down • 1s pause • drive up over mid‑foot.
  • Goblet/FFESS: If shoulders prefer, hold DB/KB at chest (or front‑foot‑elevated split squat). Same tempo and effort.
  • Not OK: shoulder pain with bar position, shrugging, wrist extension jam. Switch to Goblet/FFESS if any of these show up.
Romanian Deadlift (DB/KB or Bar) — 3×8 light–moderate
  • Soft knees; push hips back; spine neutral; lats packed; weights skim thighs; stop mid‑shin. 2s up.
  • Use straps if grip/elbow complain; neutral wrists.
Chest‑Supported Row (neutral grip) — 3×10–12
  • Bench 30–45°; chest down; palms facing. Depress/retract slightly; row elbows 30–45° from body; pause 1s; lower 2–3s. Neck long; no shrug.
  • Cranking with biceps; shoulders hiking; elbows flared to 90°.
Neutral‑Grip DB Press (slight incline) — 3×8–10
  • Bench 15–30°. DBs start above lower chest with palms facing (neutral). Elbows 30–45° from torso.
  • Lower under control (2–3s) to a pain‑free depth; press up without shrugging; ribs down; neck long.
  • Range rule: stop 1–2 cm above any pinch; use smaller ROM and increase later.
  • Elbows flared wide; rib flare/over‑arching; anterior shoulder pinch. If these appear, raise the incline or lighten the load.
Band External Rotation (ER) — 2–3×12 (slow 3s return)
  • Anchor at elbow height on opposite side. Elbow glued to side at 90°. Rotate forearm out; pause; slow 3s return.
  • Keep ribs down, shoulder blade gently set down/back, neck relaxed.
Band Internal Rotation (IR) — 3×10–12 (slow 3s return)
  • Anchor at elbow height; elbow glued at 90°. Pull forearm toward belly; pause; slow 3s return. Maintain neutral wrist.
Wrist‑Extensor HSR — 3×10–12 (2–3s up / 3s down)
  • Forearm supported, palm down, wrist just off table edge. Lift weight in 2–3s; lower in 3s through a tolerable range (≤3/10).
  • Rest 60–90s between sets. Choose a load that makes last 2 reps challenging but clean.
  • Sharp pain, tingling, or next‑day pain >3/10 → reduce ROM or load; fall back to isometrics if needed.
Optional Calm Set (Isometric) — 1×20–30s @ 3–4/10
  • Back‑of‑hand press into wall with elbow at 90°, wrist neutral or ~15° extension. Match pressure so the wrist doesn’t move; breathe.

Tuesday — Barefoot Walk–Jog + Foot/Calf Prep (~30 min)

Short‑Foot Drill — 3×20s/side
  • Gently lift arch (ball of big toe draws toward heel) without curling toes; breathe and relax the ankle.
Double‑Leg Calf Raise — 3×12 slow
  • Full range up/down 2–3s each; rise over big/second toe; brief pause at top; no bouncing.
Tibialis Raise — 2×15
  • Back to wall; heels down; lift toes toward shins; control the lower; knees soft.
Run – Week 5 Session A — 5’ walk → 6–8× (2:00 jog / 1:00–2:00 walk) → 5’ walk longer bouts
  • Cadence ~170–180; land under center; quiet feet; posture tall with slight forward lean from ankles.
  • If next‑morning calf/Achilles stiffness >60 min, repeat same volume before progressing.
Daily Shoulder Block (quick) — Scap Set • ER/IR • Wall Angels
  • Keep movements easy and smooth; stop before any pinchy range.

Wednesday — Weights + Elbow HSR (~45–50 min)

Open Books (T‑spine) — 6 reps/side
  • Rotate open; breathe out into range; knees remain stacked.
Squat — choose one — 3×6–8 @ RPE 6–7 • 3‑1‑1 tempo add +1 rep on set 1 if Monday felt easy
  • Keep bar/DB path over mid‑foot; stop 2–3 reps in reserve. Quality first; micro‑load only if reps already at top end and next‑day is clean.
Romanian Deadlift — 3×8
  • Same cues; lats packed; weights close; straps ok if elbow talks.
Chest‑Supported Row (neutral) — 3×10–12
  • Depress/retract gently; row to ribs; 2–3s lower; no shrug.
Neutral‑Grip DB Press (slight incline) — 3×8–10
  • Elbows 30–45°; neutral grip; ribs down; press smoothly. Reduce ROM if any anterior pinch.
Band ER — 2–3×12 (3s return) • Band IR — 3×10–12 (3s return)
  • Elbow glued to side; smooth, pain‑free arcs; slow returns build control.
Wrist‑Extensor HSR — 3×10–12 (2–3s up / 3s down)
  • Choose a load that challenges but stays ≤3/10 discomfort; rest 60–90s.

Thursday — Barefoot Walk–Jog + Foot/Calf Prep (~30 min)

Short‑Foot Drill — 3×20s/side
  • Lift arch gently; avoid toe curling; breathe.
Double‑Leg Calf Raise — 3×12 slow
  • Full range; 2–3s up/down; rise over big/second toe.
Tibialis Raise — 2×15
  • Back to wall; heels down; toes up under control.
Run – Week 5 Session B — 5’ walk → 6–8× (2:00 jog / 1:00–2:00 walk) → 5’ walk
  • Cadence ~170–180; quiet midfoot landings; posture tall.
  • Repeat volume if next‑morning stiffness >60 min; stop early if calves/Achilles grumble.
Daily Shoulder Block (quick) — Scap Set • ER/IR • Wall Angels
  • Quality over quantity; smooth scap motion.

Friday — Weights + Elbow HSR (~45–50 min)

Open Books (T‑spine) — 6 reps/side
  • Rotate open; breathe out into range; knees stacked.
Squat — choose one — 3×6–8 @ RPE 6–7 • 3‑1‑1 tempo optional micro‑load or +1 rep
  • Micro‑load (e.g., +2.5 kg total) only if all sets clean and next‑day was ≤3/10 after Wednesday. Otherwise add a single rep to set 1.
Romanian Deadlift — 3×8
  • Hinge cleanly; lats packed; weights close; straps ok if elbow talks.
Chest‑Supported Row (neutral) — 3×10–12
  • Depress/retract; row to ribs; 2–3s lower; no shrug.
Neutral‑Grip DB Press (slight incline) — 3×8–10
  • Maintain neutral grip; elbows 30–45°; stop shy of any pinch; breathe smoothly.
Band ER — 2–3×12 • Band IR — 3×10–12
  • Slow returns (3s); elbow glued; ribs down.
Wrist‑Extensor HSR — 3×10–12 (2–3s up / 3s down)
  • Same load/tempo rules; rest 60–90s; stop before form breaks.

Saturday — Barefoot Walk–Jog + Foot/Calf Prep (~30 min)

Short‑Foot Drill — 3×20s/side
  • Lift arch; toes relaxed; breathe.
Double‑Leg Calf Raise — 3×12 slow
  • Full range; 2–3s up/down; no bouncing.
Tibialis Raise — 2×15
  • Back to wall; heels down; slow, controlled reps.
Run – Week 5 Session C — 5’ walk → 6–8× (2:00 jog / 1:00–2:00 walk) → 5’ walk
  • Quiet landings; tall posture; easy effort.
  • Stop early if calves/Achilles grumble; repeat volume if morning stiffness >60 min.
Daily Shoulder Block (quick) — Scap Set • ER/IR • Wall Angels
  • Keep it light and pain‑free; add an extra set of ER if shoulders feel great.

Sunday — Recovery Walk + Easy Shoulder Care

Easy Walk — 10–20 min
  • Comfortable pace; breathe nasally if possible.
Gentle Stretch — Pec & forearm, 20–30s each
  • No aggressive stretching; keep it easy and non‑pinchy.
Daily Shoulder Block (quick) — Scap Set • ER/IR • Wall Angels
  • Focus on smooth scap motion; pain ≤3/10.
